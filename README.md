## TODOLIST with JAVAFX



Project ini merupakan aplikasi To-Do List berbasis Java dan JavaFX yang menerapkan konsep OOP dan arsitektur MVC. Pada bagian model, kelas baseTask dan task digunakan untuk merepresentasikan data tugas, termasuk nama, deadline, serta status selesai. Untuk kebutuhan tampilan, kelas utilitas taskFormatter menyediakan cara memformat teks tugas agar tampil rapi di antarmuka. Logika pengelolaan data berada dalam taskManager, yang berfungsi menambah, menghapus, dan menandai tugas sebagai selesai menggunakan struktur ArrayList. Seluruh fungsi ini kemudian dihubungkan dengan antarmuka grafis melalui kelas App, yang menyediakan input form, menampilkan daftar tugas dalam ListView, serta mengatur aksi pada tombol seperti Tambah, Hapus, dan Tandai Selesai. Secara keseluruhan, project ini membangun aplikasi To-Do List sederhana yang terstruktur, interaktif, dan mudah dikembangkan.
