## TODOLIST with JAVAFX



Project ini merupakan sistem manajemen tugas sederhana yang dibuat menggunakan Java untuk membantu pengguna mengelola daftar tugas mereka. Di dalamnya terdapat kelas taskManager yang berperan sebagai pengelola utama seluruh data tugas. Kelas ini menyediakan fungsi untuk menambah tugas baru lengkap dengan deadline, menghapus tugas berdasarkan indeks, serta menandai tugas sebagai selesai. Semua tugas disimpan dalam sebuah ArrayList, sehingga mudah untuk ditampilkan dan dikelola. Project ini bertujuan memberikan contoh dasar pengelolaan to-do list dengan konsep OOP dan dapat dikembangkan lebih lanjut menjadi aplikasi manajemen tugas yang lebih lengkap.
